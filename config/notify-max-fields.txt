VPN Probe — максимум полей в уведомлениях бота
==============================================

Ограничение Telegram: до 4096 символов на сообщение. Полный JSON всегда
сохраняется в Supabase (таблица probe_events). В канал уходит сжатый текст.

────────────────────────────────────────────────────────────────────────────
СОБЫТИЕ: visit (первый визит за сессию)
────────────────────────────────────────────────────────────────────────────

Геолокация (resolveLocale)
  ip                  Публичный IP
  geo_country         Страна
  geo_country_code    Код страны (RU, US…)
  geo_region          Регион / область
  geo_city            Город
  geo_lat             Широта
  geo_lon             Долгота
  geo_isp             Провайдер / ASN org
  geo_label           Сводка: город, регион, страна

Страница
  url                 Полный URL
  host                Домен
  path                Путь
  referrer            Откуда пришёл (или —)
  page_title          Заголовок вкладки

Клиент / браузер
  language            navigator.language
  languages           navigator.languages (через запятую)
  timezone            IANA timezone
  local_time          ISO 8601 UTC
  platform            navigator.platform
  user_agent          Полный User-Agent
  screen              screen.width x height
  viewport            innerWidth x innerHeight
  pixel_ratio         devicePixelRatio
  color_depth         screen.colorDepth
  online              onLine
  cookies             cookieEnabled
  cores               hardwareConcurrency
  memory_gb           deviceMemory (если доступно)
  touch               maxTouchPoints > 0
  color_scheme        dark / light
  visibility          document.visibilityState

Сеть (Network Information API, если есть)
  net_type            effectiveType (4g, wifi…)
  net_downlink_mbps   downlink
  net_rtt_ms          rtt
  net_save_data       saveData

Telegram Mini App (если открыто в Telegram)
  tg_platform         ios / android / tdesktop / web…
  tg_version          версия WebApp API
  tg_expanded         развёрнуто на весь экран
  tg_viewport_h       высота viewport TG
  tg_color_scheme     тема TG
  tg_user_id          ID пользователя (если /start был)
  tg_username         @username
  tg_name             имя из профиля

Служебное
  session_id          UUID сессии (sessionStorage)

────────────────────────────────────────────────────────────────────────────
СОБЫТИЕ: scan_complete (тест завершён)
────────────────────────────────────────────────────────────────────────────

Всё из блока «Геолокация» и «Клиент» (кроме page/referrer — дублируется
language, timezone, platform, user_agent, screen, viewport).

Время прогона
  started_at          ISO начала
  finished_at         ISO окончания
  duration_sec        Длительность, сек

Итоговые оценки
  overall_score       Общий score %
  overall_verdict     Текстовый вердикт
  censorship_likelihood  Вероятность цензуры %
  freedom_score       100 − censorship_likelihood
  split_censorship    Оценка слоя цензуры
  split_vpn_transport Оценка VPN-транспортов
  split_baseline      Оценка baseline

Статистика проб
  probes_total        Всего проб
  probes_ok           Успешных
  probes_blocked      Заблокированных
  probes_timeout      Таймаутов
  probes_error        Ошибок
  probes_inconclusive Неопределённых
  latency_p50_ms      Медиана latency
  latency_p95_ms      P95 latency
  latency_min_ms      Минимум
  latency_max_ms      Максимум

РФ
  russia_blocked_index    Индекс блокировок реестра %
  russia_blocked_total    Всего в реестре
  russia_blocked_down     Недоступно из реестра
  russia_control_total    Контрольных РФ
  russia_control_up       Контрольных доступно

Специфичные сигналы
  dns_poisoned        DNS poison / блок
  image_blocks        IMG-пробы с блоком
  throttle_ratio      Средний ratio throttling

Ошибки (error_breakdown)
  error_breakdown     dns_failure:N, timeout:N, tcp_reset:N…

Слои (layers_summary)
  layers_summary      Базовый интернет: 5/5 ок | DNS: 3/8 ок | …

Протоколы (protocols_summary)
  protocols_summary   VLESS+WS: likely_blocked (45%) | WireGuard: …

Заблокированные пробы (до 20 имён)
  blocked_probes      X (HTTPS), Instagram IMG, …

Каскад
  cascade_total       Узлов в каскаде
  cascade_ok          Узлов OK
  cascade_fail        Узлов с блоком/таймаутом

Окружение отчёта
  env_probes          Число проб в environment
  env_cascade         Число узлов каскада
